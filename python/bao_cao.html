Git là hệ thống quản lý phiên bản phân tán( quản lý mã nguồn mở)

=> teamwork, share, task, tổng hợp code

=> quản lý code và lưu trữ lịch sử code

=> dễ use, linh hoạt khi cùng làm nhiều task

Để clone repo về cần: 1. Đến trang GitHub 2. Nhấn clone or dowload =>copy url 3. Mở thư mục mình muốn chứa repo trên terminal 4. Dùng git clone để sao chép repo về máy (git lone paste url cần clone)
Lệnh git pull được sử dụng để thêm các thay đổi vào nhánh chính (git pull remote branch)
Lệnh git push được sử dụng để cập nhật các nhánh từ xa với những thay đổi mới nhất mà bạn đã commit (git push remote branch)
